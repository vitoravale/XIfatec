## Descrição

Código fonte da palestra Desenvolvimento de APIs com [Node.js](https://nodejs.org/en/) e [Nest Framework](https://github.com/nestjs/nest).

## Instalação

```bash
$ npm install
```

## Executando

```bash
# executar
$ npm run start

# executar com [nodemon](https://nodemon.io/)
$ npm run start:dev
